# JetsonNano_DT_SPI

unofficial script for Jetson Nano(L4T Jetson-Nano-Tegra210_Linux_R32.1.0_aarch64) to enable SPI1

First, downloading this repository into your Ubuntu PC (not your Jetson Nano).
After downloading, just run:

```sh
./run.sh
```
